#  People You May Know System – Data Science Project

## The Challenge

Imagine this:

You ask Sam Altman, CEO of OpenAI, for **$2.2 billion USD in funding** for your startup.

He laughs.

Then he publicly makes fun of your pitch.

After talking to you, he gives you one final challenge:

> "Collect raw Instagram data of all OpenAI followers and answer these questions within 24 hours."

### Questions to Answer

1. Who has the maximum number of posts?
2. Who has the maximum number of followers?
3. Who follows the maximum number of people?
4. How many account categories exist (Digital Creator, Non-Profit Organization, Entrepreneur, Artist, etc.)?
5. How many people belong to each category?

You ask him for the data.

He laughs again.

> "No data. Figure it out yourself."

You now have **24 hours** to:

* Collect raw Instagram data
* Clean and preprocess the dataset
* Analyze user profiles
* Build recommendation and analytics pipelines
* Generate meaningful insights from social network data

## 🎯 Project Objective

This project was developed as part of the **Data Science Course by CodeWithHarry** and explores concepts such as:

* Data Collection and Web Scraping
* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Social Network Analysis
* Recommendation Systems
* User Categorization and Statistics
* Graph-Based Thinking
